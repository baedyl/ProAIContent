## Quick Test Checklist

Before publishing, check:

- [ ] Does it use contractions? (don't, it's, you're)
- [ ] Are there questions? (at least 2-3)
- [ ] Does it address "you"? (reader directly)
- [ ] Are sentences varied? (short and long)
- [ ] Does it have personality? (opinions, emotions)
- [ ] Are there em-dashes or parentheticals?

**6/6 = Excellent humanization**  
**4-5/6 = Good, minor edits needed**  
**<4/6 = Regenerate with better settings**

## Content Type Specific Tips

### Blog Posts
- Use "conversational" tone
- Add personal anecdotes
- Include questions in each section
- Use "I" and "you" frequently

### Product Reviews
- Share "personal experience" (even if hypothetical)
- Use emotional words: love, hate, frustrating, amazing
- Include specific numbers: "after 3 weeks" not "after some time"
- Add comparisons: "better than X" or "similar to Y"

### Comparison Articles
- Use "we tested" or "we compared"
- Include opinions: "we prefer X because..."
- Add "winner" sections with reasoning
- Use casual transitions: "Now, let's look at..."

### Affiliate Content
- Write like recommending to a friend
- Use "you should" and "you'll love"
- Share "why I chose this"
- Include "honest" caveats: "it's not perfect, but..."
